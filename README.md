listen-to-me v1

so in this version you can see your current album playing on the website. it is 3D by using three.js to make up the whole environmet, camera, animations. the css is used for the buttons only.
I made a v1 so far where it displays the current album playing and the name of the song and can also switch songs. while using a cloudflare worker to send data to spotify.
in this version you are capable of:
1- switch between the next and previous song in the queue from the website itself.
2- you can see the current album picture playing 
3- you can see the name of the song playing as the website gets data every 3 seconds
